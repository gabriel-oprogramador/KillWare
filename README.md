# GT – Game Template / KillWare

**GT Engine** é um **template minimalista** para jogos 2D/3D, focado em **ECS-True** e **Data-Oriented Design (DoD)**.

O projeto é um **FPS "Infinite Runner"** onde você controla um caçador de **Wares** dentro da **Intra-Net**, uma rede virtual onde consciências se conectam.  
Se torne um **KillWare**: corra, atire e desvie de obstáculos enquanto limpa **servidores infectados** por **Wares**,  antes de ser **expulso**.

## Diferenciais
- **ECS-True & DoD:** performance máxima com estrutura de dados cache-friendly.  
- **Cross-platform:** Windows, Linux e Web (Emscripten).  
- **Minimalista:** apenas o que o jogo precisa, sem dependências externas complexas.  
- **Renderização:** OpenGL escrito do zero, sem usar bibliotecas como SDL, SFML, GLFW, GLU ou GLM.  
- **Assets:** carregamento de texturas, fontes, modelos e etc... via libs `stb_*`.  

## Ideal Para
- Quem quer estudar **ECS/DoD** na prática.  
- Base para **Games 2D/3D, eficientes e customizáveis**.
- Desenvolvimento de Games **low-level em C puro**, totalmente do zero, **sem engines externas** ou bibliotecas gráficas conhecidas.
